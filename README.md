# Sistema de Navegación Estelar - NASA

Sistema para calcular rutas óptimas entre constelaciones considerando recursos limitados.

## Características

- Visualización interactiva de constelaciones
- Cálculo de rutas óptimas
- Simulación de consumo de recursos
- Gestión del estado de salud del burro
- Algoritmos de búsqueda de caminos
